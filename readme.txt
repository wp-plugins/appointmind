== Description ==

Include your Appointmind or Schedule Organizer online appointment scheduling calender in any article or in the sidebar. This plugin requires that you have purchased either a monthly subscription or the downloadable version of the software. This plugin does not include the appointmind scheduling software. You can get the subscription or the software at <a href="http://www.schedule-organizer.de/en/?tracking=wordpress" target="_blank">Schedule Organizer</a>.

== Installation ==

1. Upload the `appointmind` folder to `/wp-content/plugins/`.
2. Activate the plugin on the 'Plugins' page in WordPress.
3. You can add your calendar URL in 'Settings' -> 'Appointmind Calendar'
